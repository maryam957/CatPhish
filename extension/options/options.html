About CatPhish

Version 1.1.0 — Security Engineering Project

  • Real-time URL risk scoring (entropy, homoglyphs, IP literals, patterns)
  • Threat-feed lookup via SHA-256 hash prefix (K-anonymity)
  • Malicious JS detection (keyloggers, skimmers, miners, obfuscation)
  • Community phishing reporting with HMAC-signed submissions
  • HMAC-chained tamper-evident audit log
  • AES-256-GCM encrypted local storage
